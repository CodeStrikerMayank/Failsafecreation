# UAV_OS | Tactical Specification Portal

A high-performance, aerodynamically themed web portal designed to showcase Unmanned Aerial Vehicle (UAV) specifications, mission roles, and strategic military classifications.

## 🚀 Overview

UAV_OS is a conceptual tactical portal built with a focus on immersive user experience, utilizing modern CSS techniques and smooth scroll-triggered animations to simulate a high-tech mission control interface.

## ✨ Key Features

- **Shutter Wipe Transitions:** Sections are revealed through a cinematic "wipe" effect using CSS `clip-path` as the user scrolls.
- **HUD Interface:** Real-time simulation of tactical stats including CPU load, latency, and signal strength.
- **Aerodynamic Profiles:** Detailed breakdown of UAV chassis types (Fixed-Wing, Multi-Rotor, etc.).
- **Strategic Classification:** Interactive data tables mapping UAV groups to operational roles and altitudes.
- **Glassmorphic UI:** A modern, semi-transparent interface with high-contrast accents and tactical typography.
- **Responsive Readability:** Optimized font scaling and line-heights for maximum legibility across all devices.

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5, CSS3 (Modern features: Flexbox, Grid, Clip-Path, Custom Properties).
- **Typography:** Inter & JetBrains Mono (via Google Fonts).
- **Interactions:** Vanilla JavaScript using `IntersectionObserver` for performant, scroll-based triggers.
- **Media:** Optimized MP4/MOV tactical footage.
